About the website:

I chose many of the design formats that I've used on this site was because i was going for a minimal design layout to make sure that the user's focus was always drawn to the game at the center of the screen. This is also why the game's background is drastically different from the background. But I knew that if I left the background blank that the overall page wouldn't be very visually interesting so I added the rainbow gradient background that I found while working on a previous project from this site:https://codepen.io/nohoid/pen/kIfto. for my above and beyond i incorporated a completely foreign javascript library that I had not used until I began working on this project I brought my knowledge from the other javascript libraries that I've used to give myself some context for this library.i also brought my prior design experience to minimize the amount of info on the screen while not sacrificing usability.
